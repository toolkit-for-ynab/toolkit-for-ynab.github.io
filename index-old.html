The Toolkit for YNAB is a free, community contributed, open source browser extension for use with You Need a Budget. It adds tweaks and additional features for a more customizable experience.

  • Contact us on Twitter, or submit a question on the YNAB Forum or on Reddit.
  • Found a bug? Report it using GitHub Issues.
  • Vote and comment on our roadmap on Trello.
  • Interested in contributing? Submit a pull request on GitHub.
