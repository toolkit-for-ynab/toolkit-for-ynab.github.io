The Toolkit for YNAB is a free, community contributed, open source Chrome browser extension to use with You Need a Budget. It adds tweaks and additional features for a more customizeable experience.

GET STARTED:

Install the Toolkit

INSTALL 1.2.0 for CHROME

Adjust the Toolkit Settings

Customize the settings to fit your needs. Save when done.

Refresh YNAB

Toolkit for YNAB automatically adds features and functionality.

The Toolkit for YNAB is not officially affiliated with or endorsed by YNAB in any way.
