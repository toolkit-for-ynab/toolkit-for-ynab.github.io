The Toolkit for YNAB is a free, community contributed, open source Chrome browser extension to use with You Need a Budget. It adds tweaks and additional features for a more customizeable experience.

How to report bugs?

Any Toolkit for YNAB extension issue can be reported to the Toolkit team using GitHub Issues. Please try to ensure the issue has not already been reported.

If you need to report a YNAB bug to YNAB itself, please first disable the Toolkit to identify whether the issue is with the extension, or with YNAB itself.

Contribute

The Toolkit is fully open source and built by many contributors. If you have the time and ability, new contributors are always welcome!

ROADMAP & NEW FEATURES:

Features under consideration and the general status of the project and roadmap is on our Trello Board. Feel free to vote and comment.

To suggest a new feature, please visit the Extension section of the YNAB Forum and create a new thread titled "Toolkit Feature Request: [a short description of the feature]". Forum user @bluebird8203 is managing the features there and will make sure it ends up in Trello.

DEVELOPMENT QUESTIONS?

View the documentation, or contact @blarg on the YNAB forums.

HOW TO SUBMIT CODE

You can see what is being worked on and by whom on our Trello roadmap. If you can't find what you want to build on the roadmap, feel free to put a note up on the GitHub issues board to let the team know you're working on something new. When your code is ready, submit a pull request.
